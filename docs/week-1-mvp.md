# Week-1 MVP

## Product cutline

This repository is intentionally optimized for a seven-day internal MVP, not a complete HR suite.

### Day 1

- boot Frappe + ERPNext + HRMS development bench
- install this `aihr` app
- create baseline roles, users, and permissions

### Day 2

- finalize `Job Requisition` custom fields
- define standard hiring request intake form for managers
- generate agency brief from requisition data

### Day 3

- enable candidate intake through `Job Applicant`
- attach resume PDFs
- run resume text extraction and heuristic screening

### Day 4

- create manager-facing candidate brief using `AI Screening`
- add shortlist and rejection operating rules
- confirm interview question template format

### Day 5

- apply interview follow-up fields
- set feedback deadlines
- add reminders and dashboards if time remains

### Day 6

- connect `Job Offer` handoff fields
- define onboarding owner and payroll handoff state
- seed real company roles and a small set of live candidates

### Day 7

- run UAT with HR and one hiring manager
- cut nonessential scope
- fix bugs and publish the internal pilot

## Non-goals for week 1

- payroll calculations
- attendance and leave
- expense claims
- full employee lifecycle automation
- public candidate portal
- deep integration with external job boards
- outbound voice agents

## Success metrics

- new role intake completed in under 15 minutes
- resume-to-brief conversion completed in under 2 minutes
- hiring manager can decide advance or reject from one screen
- no Excel is needed for the pilot roles
