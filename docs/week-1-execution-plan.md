# AIHR 7 天执行计划

## 计划周期

- 开始日期：2026-03-20
- 目标日期：2026-03-26
- 缓冲与试运行：2026-03-27

## 本周目标

在 7 天内交付一个可试运行的 AIHR 招聘 MVP，替代当前 Excel + PDF + 微信/电话驱动的核心招聘协作流程。

本周只做这条主线：

`岗位需求 -> 岗位发布 -> 简历接收 -> AI 初筛 -> 面试协同 -> Offer -> 入职交接 -> 薪酬建档准备`

## 范围边界

### 本周必须完成

- Frappe + ERPNext + HRMS 开发环境可运行
- `aihr` 自定义 app 可安装、可迁移
- `Job Requisition` 招聘需求扩展字段可用
- `Job Applicant` 简历接收和结构化字段可用
- `AI Screening` 候选人摘要卡可生成
- `Interview` 跟进字段可用
- `Job Offer` 入职和薪酬交接字段可用
- 至少 1 个真实岗位和 3-5 份简历可以跑通演示流程

### 本周明确不做

- 复杂薪资核算
- 考勤、请假、报销
- 绩效、培训
- 外部招聘网站深度 API 对接
- 电话语音机器人
- 候选人完整门户
